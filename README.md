# Spacez - Offers Page

A pixel-perfect recreation of the Spacez coupons page design in Next.js with full interactivity.

## Features

- ✅ Three interactive tabs: Coupons, Giftcards, and Payment Offers
- ✅ Sitewide coupons with Copy functionality
- ✅ Bonus gift cards with Collect buttons
- ✅ Payment offers with expandable details
- ✅ Reward vouchers section with claim functionality
- ✅ Partner offers section
- ✅ Bottom navigation bar with 5 menu items
- ✅ Success toast notifications for all interactions
- ✅ Fully responsive design
- ✅ Modern UI with Tailwind CSS
- ✅ TypeScript for type safety

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

## Interactive Elements

All buttons and interactive elements show success messages:
- **Copy buttons** - Copy coupon codes to clipboard
- **Collect buttons** - Collect gift cards
- **Read more buttons** - Expand offer details
- **Claim gift cards button** - Claim reward vouchers
- **Unlock offers button** - Access partner offers
- **Bottom navigation** - Navigate between sections

## Technology Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Lucide React (Icons)
